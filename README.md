# Violetta

An iOS client for the next-gen [Violet manga server](https://github.com/project-violet/violet/).

## Overview

Violetta is a modern, native iOS client designed to connect seamlessly to your self-hosted Violet server. It provides a fast and clean reading experience for your personal manga collection.

## Prerequisites

- iOS 18.0+
- A running instance of the [Violet server](https://github.com/project-violet/violet/)
